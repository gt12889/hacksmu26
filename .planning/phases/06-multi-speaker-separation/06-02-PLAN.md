       a. Build y_mix = 0.5 * synth_harmonic(F0_A) + 0.5 * synth_harmonic(F0_B)

       d. top_k_f0s, top_k_scores = detect_f0_shs_topk(ctx, k=2)
       e. if not is_multi_speaker(top_k_scores):
              print("Single caller detected — falling back to single-caller pipeline")
              return
       f. f0_tracks = link_f0_tracks(top_k_f0s, top_k_scores)
       g. caller_ctxs = separate_speakers(ctx, f0_tracks, output_dir, "demo")
       h. Build figure (MULTI-04):
          - `fig, ax = plt.subplots(figsize=(14, 5), constrained_layout=True)`

            - origin="lower", aspect="auto", cmap="magma", vmin=-80, vmax=0
            - extent=[0, n_frames * HOP_LENGTH / DEFAULT_SR, 0, DISPLAY_FREQ_MAX_HZ]

                ax.plot(time_axis, f0_contour, color=color, linewidth=2.5,
                        label=f"Caller {i+1}: {float(f0_contour.mean()):.1f} Hz mean f0")
          - ax.set_ylim(0, DISPLAY_FREQ_MAX_HZ)
          - ax.set_xlabel("Time (s)"), ax.set_ylabel("Frequency (Hz)")
