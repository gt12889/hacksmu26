  <behavior>
    detect_f0_shs_topk(ctx, k=2):
    - Replicates the shs_scores matrix construction from detect_f0_shs exactly
      (same F0_MIN_HZ=8.0, F0_MAX_HZ=25.0, MAX_HARMONIC_HZ=1000.0, step=hz_per_bin/2)
    - Returns top_k_f0s (k, n_frames) and top_k_scores (k, n_frames), sorted descending by score
    - Uses np.argpartition(-shs_scores, k-1, axis=0)[:k, :] then argsort to sort

    link_f0_tracks(top_k_f0s, top_k_scores, n_tracks=2, jump_hz=F0_JUMP_TOLERANCE_HZ, min_score_ratio=MIN_SCORE_RATIO):
    - Seeds first frame with top-k candidates
    - Greedy forward pass: for each track, find closest unassigned candidate within jump_hz;
      hold last value if none within tolerance
    - After linking: apply scipy.ndimage.median_filter(size=9) independently to each track
    - Returns tracks array (n_tracks, n_frames)

    is_multi_speaker(top_k_scores, min_score_ratio=MIN_SCORE_RATIO):
    - Returns True if median(top_k_scores[1] / top_k_scores[0]) >= min_score_ratio
    - Returns False otherwise (single-caller, suppress second track)

    is_harmonic_overlap(f0_a, f0_b, tolerance=0.05):
    - Returns True if abs(max(f0_a,f0_b) / min(f0_a,f0_b) - round(max/min)) < tolerance
    - i.e., ratio is within 5% of an integer

    separate_speakers(ctx, f0_tracks, output_dir, base_name):
    - For each row i in f0_tracks:
        ctx_caller = {**ctx, "f0_contour": f0_tracks[i]}
        ctx_caller = build_comb_mask(ctx_caller)
        ctx_caller = apply_comb_mask(ctx_caller)
        out_path = Path(output_dir) / f"{base_name}_caller_{i+1}.wav"
        sf.write(str(out_path), ctx_caller["audio_comb_masked"], ctx["sr"])
    - Returns list of ctx_caller dicts (one per track)

    Anti-patterns (from RESEARCH.md):
    - Do NOT run HPSS inside multi_speaker.py — ctx already has magnitude_harmonic from caller
    - Do NOT use ctx["magnitude_harmonic"] for reconstruction — build_comb_mask uses ctx["magnitude"]
    - Do NOT median-filter tracks jointly — filter each track independently
