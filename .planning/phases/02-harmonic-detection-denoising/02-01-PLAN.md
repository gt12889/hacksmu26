**Function 2 — detect_f0_shs (HARM-03, HARM-04):**
Use the VECTORIZED version from 02-RESEARCH.md "Vectorized SHS Harmonic Power Accumulation" (NOT the slow per-frame Python loop). This is the correct implementation. Then apply the octave-check per-frame after argmax:

```python
F0_MIN_HZ = 8.0
F0_MAX_HZ = 25.0
MAX_HARMONIC_HZ = 1000.0
```

Steps:
1. magnitude = ctx["magnitude_harmonic"]  (HPSS output — avoids engine harmonic bias)
2. hz_per_bin = ctx["hz_per_bin"]
3. f0_candidates = np.arange(F0_MIN_HZ, F0_MAX_HZ, hz_per_bin / 2)
4. Build shs_scores matrix (len(f0_candidates), n_frames) using vectorized pattern
5. best_candidate_idx = np.argmax(shs_scores, axis=0)
6. f0_contour_raw = f0_candidates[best_candidate_idx]
7. Apply octave-check PER FRAME (vectorized):
   ```python
   # Octave-check: HARM-04
   # For frames where detected f0 > 30 Hz, compare energy at f0/2 vs f0
   half_bins = np.round(f0_contour_raw / 2 / hz_per_bin).astype(int)
   best_bins = np.round(f0_contour_raw / hz_per_bin).astype(int)
   # Clamp to valid range
   half_bins = np.clip(half_bins, 0, n_freq_bins - 1)
   best_bins = np.clip(best_bins, 0, n_freq_bins - 1)
   # For each frame, get power at those bins (use mean across frames for vectorized access)
   # Must be done per-frame since bins differ per frame
   for t in range(n_frames):
       if f0_contour_raw[t] > 30.0:
           half_power = magnitude[half_bins[t], t]
           best_power = magnitude[best_bins[t], t]
           if half_power >= 0.7 * best_power:
               f0_contour_raw[t] = f0_contour_raw[t] / 2
   ```
8. f0_contour = median_filter(f0_contour_raw, size=9)
9. ctx["f0_contour"] = f0_contour

def process_call(
