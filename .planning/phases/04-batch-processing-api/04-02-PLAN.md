    Implement `run_batch(annotations, recordings_dir, output_dir, progress_callback=None) -> list[dict]`:
    - Pre-validate: check all `(recordings_dir / row.filename)` exist, log missing files, collect skips upfront
    - `output_dir.mkdir(parents=True, exist_ok=True)`
    - `total = len(annotations)`; wrap loop in `tqdm(annotations.itertuples(), total=total, desc="Batch")`
    - Per-row processing:
      1. `wav_path = recordings_dir / row.filename`
      2. If not wav_path.exists(): append `{"filename": row.filename, "status": "skipped", ...}`, increment progress, continue
      3. Load audio: `y, sr = load_call_segment(wav_path, row.start, row.end)`
      4. Load noise clip: `gaps = extract_noise_gaps(wav_path, [(row.start, row.end)], row.end + 30.0)`. If gaps: `noise_clip, _ = load_call_segment(wav_path, gaps[0][0], gaps[0][1], pad_seconds=0.0)`. Else noise_clip = None.
      5. Classify noise: If `hasattr(row, 'noise_type') and pd.notna(row.noise_type)`: use `{"type": str(row.noise_type).lower(), "spectral_flatness": 0.0, "low_freq_ratio": 0.0}`. Else: `noise_type_dict = classify_noise_type(noise_clip if noise_clip is not None else y[:int(1.5*sr)], sr)`.
      6. `ctx = process_call(y, sr, noise_type_dict, noise_clip=noise_clip)`
      7. Compute SNR-before: `snr_before = compute_snr_db(ctx["magnitude"], ctx["freq_bins"], float(np.median(ctx["f0_contour"])))`
      8. Compute SNR-after: `ctx_clean = compute_stft(ctx["audio_clean"], sr)`, `snr_after = compute_snr_db(ctx_clean["magnitude"], ctx_clean["freq_bins"], float(np.median(ctx["f0_contour"])))`
      9. Compute harmonic coverage: `harmonic_bins_total = int(np.sum(ctx["comb_mask"] > 0))`, `harmonic_bins_masked = int(np.sum((ctx["comb_mask"] > 0) & (ctx["masked_magnitude"] > 0)))`
      10. Compute confidence: `conf = compute_confidence(ctx["f0_contour"], snr_before, snr_after, harmonic_bins_total, harmonic_bins_masked)`
      11. Export WAV: `call_id = f"{Path(row.filename).stem}_{i:04d}"`, normalize audio, write to `output_dir / f"{call_id}_clean.wav"` with `sf.write(..., subtype="PCM_16")`
      12. Build result dict using the suffixed key names: `{"filename": ..., "start": ..., "end": ..., "f0_median_hz": float(np.median(ctx["f0_contour"])), "snr_before_db": snr_before, "snr_after_db": snr_after, "confidence": conf, "noise_type": ..., "status": "ok", "clean_wav_path": str(...)}`
      13. Append result, call `progress_callback(completed, total)` if provided
